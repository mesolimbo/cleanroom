Help

These regular expressions will be used to filter out divs with IDs or classes that match any of the patterns.

For example, if you enter "ad-" and "banner", any div with an ID or class containing "ad-" or "banner" will be removed from the sanitized page.

Each pattern should be on a new line. The extension will stop checking as soon as it finds a match for a div.
